# 项目 Skills

本目录提供可复用的本地 Skills，用于后续持续迭代缠论算法与图形渲染。

## 已提供技能

- `skills/chan-algo-tuning/SKILL.md`：笔/线段/中枢算法调优
- `skills/chan-visual-debug/SKILL.md`：KLineChart 叠加层与重绘调试

## 使用建议

- 在 Codex 中直接提及技能名，例如：
  - `请使用 chan-algo-tuning 修正线段定义`
  - `请使用 chan-visual-debug 修复中枢矩形拖动偏移`
- 若你的运行环境需要安装到全局技能目录，可将这两个技能目录复制到 `$CODEX_HOME/skills`。
